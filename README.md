# Front Sandbox

The Frontend Playground for Developers.

Front Sandbox is a premium browser-based frontend development playground built as a single self-contained `index.html` file. It includes inline HTML, CSS, and JavaScript with no build tools, no installation, and no Node.js requirement.

## Features

- HTML, CSS, and JavaScript editors
- Live iframe preview
- Autosave with localStorage
- Import/export workflows
- Starter templates
- Theme switching
- Device preview modes
- Built-in console
- Project files manager
- Library manager
- Snapshots/version history
- Share/embed helpers
- Diagnostics
- Local sandbox assistant

## Usage

Open `index.html` directly in a browser.
